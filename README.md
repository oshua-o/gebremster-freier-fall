# gebremster-freier-fall
dieser rechner ist im herbst 2025 entstanden, nachdem wir uns gefragt haben, wie gefährlich eigentlich eicheln sind, wenn sie einem auf den kopf fallen. daraufhin haben wir versucht abzuschätzen, was der $c_w$-wert von eicheln ist und wie schnell ihre terminalgeschwindigkeit ist.
aber da allgemeine rechner netter sind, haben wir jetzt einen rechner gebastelt, der alle möglichen gegenstände bis hin zum $c_w$-optimierten pinguin berechnen kann. in der simulation natürlich, damit sie einem nicht tatsächlich auf den kopf fallen.
